Icon Switch


Value: {{htmlSwitchStatus}}

Html Markup

<toggle-switch 
    ng-model="htmlSwitchStatus" 
    html="true" 
    class="switch-primary" 
    knob-label='<i class="fa fa-stop"></i>' 
    on-label='<i class="fa fa-thumbs-up"></i>' 
    off-label='<i class="fa fa-thumbs-down"></i>'> 
</toggle-switch>
Attribute Description
on-label Text value for the left side of the switch. Default value is On
off-label Text value for the right side of the switch. Default value is Off
knob-label Text value for the middle part of the switch.
tabindex Tab index for the switch. Default value is 0
html Boolean value indicating that the on-label, off-label or the knob-label contains pure html markup.
is-disabled Boolean value disabling The switch
broadcast Boolean value. If true a change event will trigger a root broadcast with the event name: ab-toggle-switch containg the value of the switch and the switch-id value
switch-id Unique id for the switch. Default is 0.
on-change Callback function triggered on a switch change.
