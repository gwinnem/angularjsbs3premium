With Progressbar

          boxsolid="bg-yellow"
          boxtitle="Notifications"
          boxtitletype="text"
          boxmessage="28"
          boxmessagetype="bold"
          boxicon="fa fa-envelope-o"

Output

Options

  • CSS styles is located in: content/less/plugins/info-box.less
  • Info box width is fixed at 100%
  • Info box min-height is 90px
  • Info box margin-bottom is 15px
Attribute Description
boxbg Color used as background for the icon.
Available Colors
boxsolid Color used as background for the box.
Available Colors
boxicon Any css class representing a icon.
boxtitle Boxt title text transformed to uppercase
boxtitletype Defines if text is normal or bold.
Values:
  1. text
  2. bold
boxmessage Text displayed as content of the InfoBox
boxmessagetype Defines if text is normal or bold.
Values:
  1. text
  2. bold
boxprogress Numeric value representing the width of the progressbar.
boxprogresstext Text below progressbar.

Html markup

<div class="info-box bg-aqua">
    <span class="info-box-icon"><i class="fa fa-bookmark-o"></i></span>
    <div class="info-box-content">
        <span class="info-box-text">Bookmarks</span>
        <span class="info-box-number">41,410</span>
        <div class="progress">
            <div class="progress-bar" style="width: 70%"></div>
        </div>
        <span class="progress-description">70% Increase in 30 Days</span>
    </div>
</div>
