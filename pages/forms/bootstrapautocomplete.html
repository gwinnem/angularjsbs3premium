For simple autocomplete use cases there seems to be nothing wrong with the dropped typeahead plugin.
Here you will find the typeahead autocomplete plugin for Twitter's Bootstrap 3.
The original code is written by @mdo and @fat.

Basic example inline data

Basic example via data attributes: (type 'item...')


<input type="text"
    data-provide="typeahead"
    data-source='["item 1","item 2","item 3"]'
    placeholder="item..."
    class="form-control" />

Basic example javascript

Basic example via data attributes: (type 'item...')


$('.typeahead_1').typeahead({
    source: [
        { id: "someId1", name: "Display name 1" },
        { id: "someId2", name: "Display name 2" }
    ]
});
