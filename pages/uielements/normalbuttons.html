Normal Color buttons

Base class: .btn Each buttons has a tooltip displaying the class used.

Disabled buttons

Disabled: .disabled

Different sized buttons.

Different button sizes. .btn-lg .btn-sm .btn-xs

Application Buttons

Add the classes .btn.btn-app to an <a> tag to achieve the following: 

<a class="btn btn-app">
    <i class="fa fa-edit"></i> Edit
</a>
Edit Play Repeat Pause Save Notifications Products Users Orders Inbox Likes

Buttons with badges

Add a <span class="badge">3</span> 

<a class="btn btn-app">
    <span class="badge bg-yellow">3</span>
    <i class="fa fa-bullhorn"></i> Notifications
</a>
3 Notifications 300 Products 891 Users 67 Orders 12 Inbox 531 Likes
