ABAdmin fit-height Directive

Usage: Add the directive to any html element to fit the height of that element to the page height.

Usage:

<!-- box -->
<div id="abfitheight" class="box box-white" fit-height>
    <div class="box-header with-border">
        <h3 class="box-title">ABAdmin fit-height Directive</h3>
    </div>
    <!-- box-body -->
    <div class="box-body">
    </div>
    <!-- / box-body -->
    </div>
</div>
<!-- / box -->
