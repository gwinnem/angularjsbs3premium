The box tools directive is responsible for the toolbar in the box. Each button has the title filled out to get a tooltip.

Markup:

<box-tools 
    start-collapsed="0" 
    hide-refresh="0"
    refresh-callback="refreshFunction()"
    hide-max-min="0" 
    hide-full-screen="0" 
    hide-remove="0">
</box-tools>

Options

Parameter Description
start-collapsed Controls if the box is collapsed when it is loaded.
hide-refresh Hides or shows the refresh button.
refresh-callback The function which is called when the refresh button is clicked. The responsive from the callback is added to the box body.
hide-max-min Hides or shows the max min button.
hide-full-screen Hides or shows the full screen button.
hide-remove Hides or shows the remove button.

The parameters are optional and the Default value is false.
