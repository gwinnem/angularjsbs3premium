The box component is the most used compnent in the template. And is being used to displaying everything from charts to page content.

Html Markup:

<!-- box -->
    <div id="box" class="box box-info">
        <div class="box-header with-border">
            <h3 class="box-title">Box Title</h3>
        </div>
        <!-- box-body -->
        <div class="box-body">
            <h4>The body of the box</h4>
        </div>
        <!-- / box-body -->
        <!-- box footer -->
        <div class="box-footer">
            <h4>The footer of the box</h4>
        </div>
        <!-- / box footer -->
    </div>
<!-- / box -->
