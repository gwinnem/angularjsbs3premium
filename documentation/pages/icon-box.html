Examples

<icon-box boxbg="bg-navy"
          boxicon="fa fa-user"
          boxmessage="199">
</icon-box>

Options

  • CSS styles is located in: content/less/plugins/icon-box.less
  • height is 70px
  • width is defined by the outer html
Attribute Description
boxbg Color used as background.
Available Colors
boxicon Any css class representing a icon.
boxmessage Text displayed as content of the right side.

Html markup

<div class="icon-box bg-gray">
    <span class="icon-box-icon">
        <i class="fa fa-user"></i>
    </span>
    <div class="icon-box-text">
        <span>199</span>
    </div>
</div>
199
