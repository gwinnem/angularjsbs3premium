Examples

Without boxsref

<small-box boxbg="bg-aqua"
           boxtitle="210"
           boxmessage="New Orders"
           boxicon="fa fa-shopping-cart"
           boxfooter="More Info">
</small-box>

Output

With boxsref

<small-box boxbg="bg-aqua"
           boxtitle="210"
           boxmessage="New Orders"
           boxicon="fa fa-shopping-cart"
           boxsref="widgets"
           boxfooter="Click Me">
</small-box>

Output

Options

  • CSS styles is located in: content/less/plugins/small-box.less
  • min-width is 134px
  • min-height is 128px
  • margin-bottom is 20px
Attribute Description
boxbg Color used as background.
Available Colors
boxicon Any css class representing a icon.
boxtitle Boxt title text
boxmessage Text displayed as content of the SmallBox
boxfooter Text displayed in the footer of the SmallBox
boxsref ui-sref value for the footer text

Html markup

<div class="small-box bg-aqua">
    <div class="small-box-inner">
        <h3>150</h3>
        <p>New Orders</p>
    </div>
    <div class="small-box-icon">
        <i class="fa fa-shopping-cart"></i>
    </div>
    <a href="javascript:void(0)" class="small-box-footer">More info <i class="fa fa-arrow-circle-right"></i></a>
</div>
