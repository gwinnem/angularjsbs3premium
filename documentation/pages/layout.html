Tip!

The starter page is a good place to start building a page. The index.html file contains the css and js files needed.

The page layout ( index.html ) consists of five major parts:

  • Wrapper .wrapper. A div that wraps the whole page.
  • Main Header .main-header. Contains the logo and navbar.
  • Sidebar .main-sidebar. Contains the user panel and sidebar menu.
  • Content .content-wrapper. Contains the page header and content.
  • Footer .main-footer. Contains the page footer.

Layout Options

Note!

You cannot use both layout-boxed and fixed at the same time. Anything else can be mixed together.

AbAdmin provides a set of options to apply to your main layout. Each one of these classes can be added to the body tag to get the desired goal.

  • Fixed: use the class .fixed to get a fixed header and sidebar. Main content not scrollable
  • Collapsed Sidebar: use the class .sidebar-collapse to have a collapsed sidebar upon loading.
  • Boxed Layout: use the class .layout-boxed to get a boxed layout that stretches only to 1250px.
  • Top Navigation use the class .layout-top-nav to remove the sidebar and have your links at the top navbar. Not working
