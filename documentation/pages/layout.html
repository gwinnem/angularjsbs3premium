Layout Options

Note!

You cannot use both layout-boxed and fixed at the same time. Anything else can be mixed together.

The template depends on the following files to make the layout work correct.

File Description
abTree.js Handles sidebar treeview menu.
abLayout.js The main layout handler. For layout options fixed and layout-boxed.
abPushMenu.js For layout options fixed and layout-boxed.
abControlSidebar.js For righ side menubar/control sidebar.

AbAdmin provides a set of options to apply to your main layout. Each one of these classes can be added to the body tag to get the desired goal.

  • Normal: use the class .layout-normal to get the normal layout.
  • Fixed: use the class .layout-fixed to get a fixed header and sidebar. Main content not scrollable
  • Boxed Layout: use the class .layout-boxed to get a boxed layout that stretches only to 1250px.
  • Collapsed Sidebar: use the class .sidebar-collapse to have a collapsed sidebar upon loading.
  • Control Sidebar Open: use the class .controlsidebar-open to have the control sidebar open upon loading.
  • Control Sidebar Slide Open: use the class .controlsidebar-slide-open Control sidebar push content instead of sliding over content..
  • Fixed Footer use the class .set-main-footer-fixed to make the footer stick at the bottom of the screen. Does not work with boxed layout.
